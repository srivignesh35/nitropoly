INSTALLATION


  1. Delete old "Nitropoly.py" file.
    (Ignore this step 1 if you are installing freshly)
  2. Paste/overwrite the "NitroPoly.py" into your Maya Script Folder.
    Something looks like this "Documents\maya\2018\scripts"
  3. Open the script Editor , paste and save the below mentioned Python command into you shelf.
    Finally run the tool.

import NitroPoly;reload(NitroPoly);NitroPoly.main()
